## APP LAUNCH
### 1. Clone repository
`git clone https://github.com/davidrepository/live-football-scoreboard-oo.git`

### 2. Go to `/server` folder
`cd server`

### 3. Install dependencies
`npm install`

### 4. Run app
`npm run dev`


## APP TESTING
### 1. Go to `/server` folder
`cd server`

### 2. Run tests
`npm run test`
